## Switch Contract
### Install
```sh
yarn
```
### Cmd
compile
```sh
yarn compile
```
test
```sh
yarn test
```
deploy

```sh
# scripts/deploy.ts
yarn deploy
```

```sh
# scripts/verify.ts
yarn verify
```

```sh
npx hardhat clean
```
